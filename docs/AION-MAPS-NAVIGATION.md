# AION как навигатор (карта уровня Waze/Google) — план

**Статус: PLANNED (частично уже работает).** Вопрос владельца: может ли AION быть
настоящей картой — как Waze/Google Maps — которую можно подключить к приложению
Bolt, со своими маршрутами и точками.

**Короткий ответ: да, поэтапно.** База уже есть; полноценная навигация — большой,
но реальный путь. Ничего не выдумываем: карта — OpenStreetMap, маршруты — только
по реальным дорогам (OSRM), точки — только реальные (GPS смены, АЗС, адрес из Bolt).

## Что уже работает сегодня (после OTA)

- Своя карта на OpenStreetMap (без Google-ключей): экран «Карта Driver».
- Заправки рядом (реальные, из OSM) с подсказкой «дешевле рядом».
- **Маршрут смены на карте**: активная смена рисуется по GPS вживую; без активной
  — показывается маршрут последней смены (старт/финиш точками).
- Выдуманные «горячие зоны» с карты УДАЛЕНЫ (были фейковые круги — нарушение
  принципа честных данных).

## Как AION подключается к Bolt (этап 1 — приём точки)

Bolt при нажатии «Навигация» отдаёт точку назначения системе через стандартный
Android-интент (`geo:lat,lng` / `google.navigation:q=`). Любое приложение,
объявившее intent-filter на эти схемы, появляется в списке выбора навигатора.

- Что нужно: intent-filter в AndroidManifest (нативная правка → пересборка APK)
  + экран «Точка назначения» (принять координаты, показать на карте).
- Решение владельца: пересборка APK.

## Маршрут до точки (этап 2)

Построение маршрута по дорогам — движок маршрутизации:
- **Self-host OSRM + OSM-экстракт Эстонии** — уже РЕКОМЕНДОВАН в
  [ADR-road-matching](./ADR-road-matching.md) для привязки треков; тот же сервер
  даёт и маршруты A→B. Одна инфраструктура закрывает обе задачи.
- Быстрый старт без сервера: публичный OSRM demo-сервер (лимиты, не для прод).
- Выход: линия маршрута на карте + расстояние/время из реального графа дорог.

## Полная навигация «как Waze» (этап 3 — большой)

Turn-by-turn подсказки (голос «через 200 м направо»), перестроение маршрута при
отклонении, полосы/камеры — это серьёзный объём. Реалистичный путь: OSRM отдаёт
шаги маршрута (текст манёвров), их можно озвучивать через существующий
голосовой слой. Пробки/камеры — данных нет и выдумывать их нельзя.

## Порядок

1. **Сейчас (OTA)**: маршрут смены на карте — сделано.
2. **APK-пересборка**: intent-filter → AION появляется в выборе навигатора Bolt,
   принимает точку, показывает её на карте. (Вместе со слушателем уведомлений
   из [AUTO-ORDER-CAPTURE](./AUTO-ORDER-CAPTURE.md) — одна пересборка на оба.)
3. **OSRM-сервер**: маршрут до точки по дорогам (+ road matching из ADR).
4. **Turn-by-turn**: манёвры голосом — после 2-3 и решения владельца.
