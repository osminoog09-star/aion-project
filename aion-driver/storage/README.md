# Storage (AION)

Ключи AsyncStorage **не переименовывались** — совместимость с уже установленными билдами.

## Структура

| Папка | Назначение |
|-------|------------|
| `storage/core/` | OTA (snooze, тестовые флаги), отладочный flush синка, глобальные `STORAGE_KEYS` |
| `storage/driver/` | Профиль смены, активная смена, история смен, OCR-импорты |

## Миграции

При добавлении нового модуля заводите подпапку `storage/<module>/` и константы ключей в `storage/core/keys.ts` (или отдельном файле модуля), не меняя существующие значения ключей без миграции данных.
